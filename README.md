# Node and Express workshop

## Running Locally

```sh
$ git clone git@github.com:kjarnet/webrammeverk-express.git
$ cd webrammeverk-express
$ npm install
$ npm start
```

Your app should now be running on [localhost:3000](http://localhost:3000/).


## Running Tests

```sh
$ git clone git@github.com:kjarnet/webrammeverk-express.git
$ cd webrammeverk-express
```

## Deploying to Heroku

```
$ heroku create
$ git push heroku master
$ heroku open
```
